How the AI Recommender Works

A conversational AI pipeline that maps your hiring context to the most relevant SHL assessments — using retrieval, grounded generation, and structured outputs.

The Conversation Flow

1

You describe your hiring need

Start by telling the agent who you are hiring. You can be vague ("I need a test for a developer") or specific ("Mid-level Java developer with a focus on problem-solving and personality fit").

2

The AI clarifies when needed

If your request is too vague, the agent will ask a targeted clarifying question — like the seniority level, skills required, or specific test type. It never guesses when it doesn't have enough information.

3

Catalog retrieval using TF-IDF search

Keywords extracted from your conversation are used to search the SHL product catalog using a TF-IDF similarity algorithm. The top 20 most relevant assessments are retrieved instantly, without any external API calls.

4

Grounded recommendations are generated

The AI generates its final response using only the retrieved catalog items. It cannot hallucinate or recommend products that don't exist in SHL's catalog. Refine, compare, and adjust — the agent will update its recommendations based on your feedback.
